测试Base URL配置

测试目标：
验证所有AI模型是否正确使用硬编码的base URL: https://api-dev.718ai.cn/v1

测试步骤：

  1. 设置用户信息：自动设置测试用户信息
  2. 发起对话：向AI模型发送测试消息
  3. 监控网络：检查请求是否发送到正确的base URL
  4. 验证响应：确认收到来自中转服务的响应

预期结果：

正确的请求URL: https://api-dev.718ai.cn/v1/chat/completions
错误的请求URL: https://api.openai.com/v1/chat/completions
验证方法: 打开浏览器开发者工具的Network标签页监控请求
配置说明: baseURL设置为 https://api-dev.718ai.cn/v1，SDK会自动添加具体的endpoint路径

手动验证方法：

1. 打开浏览器开发者工具 (F12) 2. 切换到 Network 标签页 3. 设置用户信息并登录LibreChat 4. 发送一条消息给AI 5. 在Network标签页中查找chat/completions请求 6. 检查请求的URL是否为: https://api-dev.718ai.cn/v1/chat/completions
